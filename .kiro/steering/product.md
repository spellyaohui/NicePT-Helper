# 产品概述

NicePT Helper 是一个针对 NicePT（基于 NexusPHP）的 PT 自动化管理系统。

## 核心功能

- PT 账号管理（Cookie 认证、数据刷新）
- 种子搜索与下载（HTML 页面解析）
- 规则引擎自动下载（促销、大小、关键词等条件）
- 下载器集成（qBittorrent / Transmission）
- 下载历史与生命周期管理
- 自动删种（促销过期、容量阈值、unregistered）
- 定时任务调度
- 仪表盘与日志管理

## 设计原则

- 站点适配解耦：site_adapter.py 封装所有站点交互
- 下载器适配解耦：统一接口支持多种下载器
- 规则与执行分离
- 防风控：请求限速、重试退避

## 当前阶段

后端 API 全部完成（25+ 接口），React 前端已搭建完成（9 个页面），前后端联调通过。

## 页面清单

登录、仪表盘、账号管理、种子搜索、规则管理、下载器管理、下载历史、系统设置、日志查看
